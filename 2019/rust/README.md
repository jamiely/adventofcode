Test
====

Run all the tests:

```bash
cargo test
```

Run individual tests for a day:

```bash
cargo test --doc day4
```

Usage
=====

```bash
scripts/run_all_examples
cat output/*
```

or individually:

```bash
cargo run --example day1
cargo run --example day2
cargo run --example day3
cargo run --example day4
# ...
```
